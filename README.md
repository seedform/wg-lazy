wg-lazy - A Simple Configuration Manager For WireGuard
===

**WARNING. THIS PROJECT IS IN ACTIVE DEVELOPMENT. DO NOT USE IT IN A PRODUCTION ENVIRONMENT OR ON ANY CRITICAL INFRASTRUCTURE.**

## Features

1. Generate WireGuard configuration files for `wg-quick`.
2. Manage peers.
3. Quickly allow peers to connect via QR code.
4. Does not require any background processes to be continuously running.



## Core Philosophies:

1. Design for testability.
2. Fail fast.
